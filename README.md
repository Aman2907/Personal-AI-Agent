# 🤖 Personal AI Agent

An AI-powered personal assistant built with Next.js, React, TypeScript, Google Antigravity 2.0 CLI, and Sent.dm.

The application enables users to interact with an intelligent AI agent capable of understanding natural language, executing tasks, maintaining conversation history, and integrating with external services.

## ✨ Features

- AI-powered personal assistant
- Natural language conversations
- Context-aware responses
- User authentication
- Chat history
- Responsive UI
- Real-time notifications
- AI Agent workflow
- Tool integrations
- Modern Next.js App Router architecture



## 🛠 Tech Stack

### Frontend

- Next.js
- React
- TypeScript
- Tailwind CSS

### Backend

- Next.js API Routes

### AI

- Google Antigravity 2.0 CLI

### Notifications

- Sent.dm

### Authentication
